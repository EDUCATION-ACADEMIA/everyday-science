A scientist observed a cell using an electron microscope. The size of the image was 25 mm. The magnification was × 100 000 Calculate the real size of the cell.

(A) 0.000 25 micrometres

(B) 25 micrometres

(C) ** 0.25 micrometres

(D) 250000 micrometres

EXPLANATIONS BELOW

Concept note-1: -Divide the number of cells in view with the diameter of the field of view to figure the estimated length of the cell. If the number of cells is 50 and the diameter you are observing is 5 millimeters in length, then one cell is 0.1 millimeter long. Measured in microns, the cell would be 1, 000 microns in length.

Concept note-2: -Actual size = image size / magnification. Image size = magnification x actual size.

Concept note-3: -Estimating the Size of the Specimen Under Observation Remember that 1 m = 0.001 mm. To estimate the size of an object seen with a microscope, first estimate what fraction of the diameter of the field of vision that the object occupies. Then multiply the diameter you calculated in micrometers by that fraction.

Concept note-4: -Calculation of Actual Size: To calculate the actual size of a magnified specimen, the equation is simply rearranged: Actual Size = Image size (with ruler) ÷ Magnification.
