Balance this equation ____ CH4 + ____ O2 → __ CO2 + ____ H2O

(A) 1, 2, 1, 1

(B) 2, 1, 2, 1

(C) ** 1, 2, 1, 2

(D) 0, 2, 0, 2

EXPLANATIONS BELOW

Concept note-1: -To balance the hydrogen atoms, put a 2 in front of H2O: CH4 + O2 CO2 + 2H2O. The oxygen atoms can be balanced by putting a 2 in front of the O2 on the left: CH4 + 2O2 CO2 + 2H2O. The equation is now balanced.

Concept note-2: -The complete and balanced chemical equation of the combustion reaction of Methane is written as: CH 4 ( g ) + 2 O 2 ( g ) → CO 2 ( g ) + 2 H 2 O ( l ) + Heat ( Methane ) ( Oxygen ) ( Carbon ( Water ) dioxide )

Concept note-3: -CH 4 ( g ) Methane + 2 O 2 ( g ) Oxygen gas → CO 2 ( g ) Carbon dioxide + 2 H 2 O ( g ) Water vapours.
