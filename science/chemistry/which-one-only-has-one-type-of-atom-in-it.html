which one only has one type of atom In it.

(A) ** Element

(B) compound

EXPLANATIONS BELOW

Concept note-1: -If a substance contains only one kind of atom, no different kind of atoms is not present in it then it is called an element. The properties of elements are: The building block of an element is atoms. For example, the silver element contains only one kind of atom of silver only.

Concept note-2: -When a substance contains only one type of atom, it is called an element. Each element has a special name and unique properties that are different from all the rest of the elements. All known elements are organized on a chart called the periodic table of elements.

Concept note-3: -A set which contains only one element is called a singleton set.

Concept note-4: -Compound: more than one kind of atom in a fixed ratio, e.g., H2O, CO, etc. Not all molecules are compounds, e.g., O2. Not all compounds are molecules because solid compounds have a distinct type of bonding, e.g., NaCl.

Concept note-5: -Elements. An element is a pure substance which is made from only one type of atom. Everything in the universe contains the atoms of one or more elements. The atoms in one element are all the same as each other, but they are different from the atoms of any other elements.
