Which electronic structure is correct for an element with 17 electrons?

(A) 1, 7

(B) 10, 7

(C) ** 2, 8, 7

EXPLANATIONS BELOW

Concept note-1: -The atomic number of Chlorine(Cl) is 17 and the electronic configuration is; 2, 8, 7 with seven electrons in the valence shell.

Concept note-2: -The neutral atom chlorine (Z=17), for instance has 17 electrons. Therefore, its ground state electronic configuration can be written as 1s22s22p63s23p5. The chloride ion (Cl-), on the other hand, has an additional electron for a total of 18 electrons.

Concept note-3: -The element here is Chlorine having an atomic number equal to 17.

Concept note-4: –Chlorine has the atomic number 17.-It belongs to group 17 in the periodic table.-The electronic configuration of chlorine is 1s22s22p63s23p5.

Concept note-5: –So, the number of electrons present in an atom = 2+8+7 = 17.-Therefore the atomic number of the element is equal to 17.-The element having atomic number 17 is Chlorine.
