explain the importance of the metric system

(A) the metric system is important because it is the system of international units and is hard and changing

(B) ** the metric system is important because it is the international system of units and it is simple and reliable

EXPLANATIONS BELOW

Concept note-1: -The metric system has been adopted by most countries because it is an easy system to use. Everything is based on the number ten. This is called a decimal system. Basically, all you need to do in order to change numbers to larger or smaller units is multiply or divide them by ten.

Concept note-2: -Scientists all over the world use the metric system. There’s a very good reason for this-it’s so everyone is doing the measuring the same way, all over the world. Most other countries already use the metric system for measuring everything.

Concept note-3: -The metric system is a system of measurement that uses the meter, liter, and gram as base units of length (distance), capacity (volume), and weight (mass) respectively. To measure smaller or larger quantities, we use units derived from the metric units.

Concept note-4: -The metric system is defined as the system of measurements to calculate the mass, distance, and volume of any object. We generally use the metric system to measure smaller or larger quantities.
