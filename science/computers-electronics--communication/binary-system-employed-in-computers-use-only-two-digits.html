Binary system employed in computers use only two digits.

(A) 1 and 2

(B) 0 and 10

(C) ** 0 and 1

(D) 1 and 10

EXPLANATIONS BELOW

Concept note-1: -Binary describes a numbering scheme in which there are only two possible values for each digit–0 or 1–and is the basis for all binary code used in computing systems.

Concept note-2: -1. Binary is a base-2 number system invented by Gottfried Leibniz that’s made up of only two numbers or digits: 0 (zero) and 1 (one). This numbering system is the basis for all binary code, which writes digital data such as the computer processor instructions used with your devices every day.

Concept note-3: -Solution: The 0 and 1 in the binary numbering system are called binary digits or Bits. A bit is the smallest unit of data in a computer. A bit has a single binary value, either 0 or 1.

Concept note-4: -binary number system, in mathematics, positional numeral system employing 2 as the base and so requiring only two different symbols for its digits, 0 and 1, instead of the usual 10 different symbols needed in the decimal system.

Concept note-5: -Computers today use digits to represent information-that’s why they’re called digital systems. The simplest and most common way to represent digits is the binary number system, with just two digits (usually written as 0 and 1). It is called binary because there are only two different digits used, or two states.
